# How to install the node

## 1 - Requisites
- Raspberry Pi 2, 2 B+ or 3 with [Raspbian](https://www.raspberrypi.org/downloads/raspbian/)
- Internet connection
- [DHT22](https://www.adafruit.com/product/385)
- [MCP3008](https://www.adafruit.com/product/856)
- Pyranometer
