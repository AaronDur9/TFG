# How to install the node

## 1 - Requisites
- Internet connection

## 2 - Files

Copy this folder into your server directory

## 3 - Dependencies

- Update
```
sudo apt-get update
sudo apt-get upgrade
```

- Python2.7
```
sudo apt-get install python2.7 build-essential python-pip python-dev
```

- Paho Mqtt: `pip install paho-mqtt`
